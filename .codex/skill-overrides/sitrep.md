## Actwyn Situation Report Sources

- Prefer `docs/context/current-state.md` for the compressed current position.
- Treat `docs/04_IMPLEMENTATION_PLAN.md` as the canonical roadmap/status ledger.
- Summarize roadmap position using Actwyn's planning terms when present:
  milestone, track, phase, and slice.
- When useful, report the current position in the form
  ``P0-M5 / OPS / OPS-1A / P0-M5.GATE`` with the current gate status.
